# 🌐 Modern Web Interface

This is a **sample modern web interface** built using **HTML and CSS**.  
The project focuses on creating a **clean, classy, and modern UI design** with a navigation bar and hover effects.

## Features
- Modern navigation bar
- Clean and responsive layout
- Smooth hover effects
- Simple and attractive UI design

## Technologies Used
- HTML
- CSS

## Project Purpose
This project demonstrates **basic UI design and layout structure** for a modern website interface.

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
